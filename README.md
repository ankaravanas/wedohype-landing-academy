# WEDOHYPE AI Landing Page Academy

Landing page για το **AI Landing Page Course** της WEDOHYPE - ένα course που διδάσκει πώς να δημιουργείς landing pages με AI εργαλεία.

## Tech Stack

- **Framework:** Next.js 16 με App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Project Structure

```
src/
├── app/           # Next.js App Router pages
├── components/
│   ├── sections/  # Page sections (Hero, FAQ, etc.)
│   └── ui/        # Reusable UI components
├── lib/           # Utilities
└── public/images/ # Static assets
```

## Deployment

Το project κάνει deploy στο GitHub Pages μέσω του `gh-pages` branch.
